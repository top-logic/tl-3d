Syntax

threejsCp(
  tx: $tx,
  classifiers: $classifiers
)

Description

Creates a connection point that can connect to other points with overlapping classifiers

Parameters

name Type Type Description Mandatory Default
tx Transformation Position of the connection point yes
classifiers List of character strings Classification of the connection point. Connections can only be made to points where at least one common classifier exists. no

Return value

Type: Connection point
