Syntax

threejsGroup(
  tx: $tx,
  userData: $userData,
  hidden: $hidden,
  color: $color,
  components: $components
)

Beschreibung

Erstellt einen Szenenknoten der andere Knoten zusammenfasst. Wird z.B. dieser Knoten verschoben werden alle inneren Knoten mitverschoben.

Parameter

Name Typ Beschreibung Pflicht Default
tx Transformation Transformation die auf den Knoten angewandt wird. Die Transformation is relativ zu der umgebenen Knotengruppe. nein
userData Objekt Fachobjekt für dem Knoten. Dieses Objekt wird z.B. für das Cross-Highlighting benutzt nein
hidden boolean Ob der Knoten dargestellt werden soll. nein false
color Color Farbe in der der Knoten gefärbt werden soll. nein
components Liste von Szenenknoten Knoten die diese Gruppe enthält nein

Rückgabewert

Typ: Szenenknoten

Ein Knoten der die gegebenen components zusammenfasst.
