Syntax

threejsImage(
  userData: $userData,
  data: $data,
  imageID: $imageID
)

Beschreibung

Ein Bild Objekt das die eigentlichen Bilddaten dynamisch abruft.

Parameter

Name Typ Beschreibung Pflicht Default
userData Objekt Das Objekt aus dem die Bilddaten ausgelesen werden ja
data Funktion Funktion die das userData-Objekt bekommt und die Bilddaten ausrechnet. ja
imageID String ID für das Bild. Haben mehrere Bildobjekte die selbe imageID so wird das bild nur einmal abgerufen. nein wird aus dem userData Object ausgerechnet

Rückgabewert

Typ: Bilddatenobjekt

Beispiele

Beispiel 1

// Part has an image attribute `image`

part = ....;
threejsImage(
  userData: $project
  data: x -> $x.get(`image`)
);

Ausgabe: Dynamisches Bild, dass die eigentlich Bilddaten aus dem project aus dem Attribut image ausliest.
