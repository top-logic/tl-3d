Syntax

$object.threejsAccess($property)

Beschreibung

Reflection-Zugriff auf das Objekt.

Parameter

Name Typ Beschreibung Pflicht Default
object ein 3D Objekt Objekt von dem Informationenbenötigt werden. ja
property String Name der Eigenschaft auf die zugegriffen werden soll. ja

Rückgabewert

Typ: Objekt

Beispiele

Beispiel 1

tx = ...;
cp = threejsCP(tx: $tx, classifiers: "important);

cp.threejsAccess("tx") == $tx; // true

Ausgabe: true
