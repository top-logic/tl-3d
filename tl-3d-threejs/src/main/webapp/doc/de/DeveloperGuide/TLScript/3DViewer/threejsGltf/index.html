Syntax

threejsGltf(
  tx: $tx,
  userData: $userData,
  hidden: $hidden,
  color: $color,
  layoutPoint: $layoutPoint,
  snappingPoints: $snappingPoints,
  url: $url,
  imageData: $imageData
)

Beschreibung

Erstellt einen Szenenknoten der ein Bild darstellt.

Parameter

Name Typ Beschreibung Pflicht Default
tx Transformation Transformation die auf den Knoten angewandt wird. Die Transformation is relativ zu der umgebenen Knotengruppe. nein
userData Objekt Fachobjekt für dem Knoten. Dieses Objekt wird z.B. für das Cross-Highlighting benutzt nein
hidden boolean Ob der Knoten dargestellt werden soll. nein false
color Color Farbe in der der Knoten gefärbt werden soll. nein
layoutPoint Verbindungspunkt Punkt innerhalb des Knotens der für das Layouten benutzt wird. nein
snappingPoints Liste von Verbindungspunkt Punkte an denen andere Element angedockt werden können nein
url String URL relative zur Applikation wo das Bild abgerufen werden kann. nein
imageData Dynamisches Bild Bild dessen Inhalt dyanmisch berechnet wird, z.B. durch Zugriff auf ein Binär-Attribut nein

Rückgabewert

Typ: Szenenknoten
