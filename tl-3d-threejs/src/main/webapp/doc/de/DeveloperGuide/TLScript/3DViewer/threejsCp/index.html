Syntax

threejsCp(
  tx: $tx,
  classifiers: $classifiers
)

Beschreibung

Erstellt ein Verbindungspunkt der mit anderen Punkten mit überschneidenen Classifiers eine Verbindung eingehen kann

Parameter

Name Typ Beschreibung Pflicht Default
tx Transformation Position des Verbindungspunktes ja
classifiers Liste von Zeichenketten Klassifizierung des Verbindungspunktes. Es können nur Verbindungen zu Punkten eingegegangen werden, wo mindestens ein gemeinsamer Klassifikator existiert. nein

Rückgabewert

Typ: Verbindungspunkt
