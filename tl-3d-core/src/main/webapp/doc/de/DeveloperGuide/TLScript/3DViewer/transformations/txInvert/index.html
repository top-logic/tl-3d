Syntax

$tx.invert()

Beschreibung

Erstellt eine Transformation die einen gegebene Transformation invertiert

Parameter

Name Typ Beschreibung Pflicht Default
tx Transformation Eine Transformation deren Wirkung invertiert werden soll. ja

Rückgabewert

Typ: `tl.threed.core:Transformation`

Beispiele

Beispiel 1

txIdentity().invert()

Ausgabe: `txIdentity()`

Die Einheitstransformation is zu sich selbst inverse.

Beispiel 2

translate(100, 100, 50)

Ausgabe: translate(-100, -100, -50)

​

Beispiel 3

rotate(Math.PI, Math.PI, Math.PI/2)

Ausgabe: rotate(z: Math.PI/2)
