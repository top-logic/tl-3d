Syntax

rotate(
  x: $xAngle,
  y: $yAngle,
  z: $zAngle
)

Beschreibung

Erstellt einen Rotationsmatrix die um die gegebenen Werte rotiert, zuerst um die z-Achse, dann um die y-Achse und dann um die x-Achse

Parameter

Name Typ Beschreibung Pflicht Default
x double Grad in Bogenmaß um den um die x-Achse rotiert werden soll: x \in [0, 2*\pi] nein 0
y double Grad in Bogenmaß um den um die y-Achse rotiert werden soll: y \in [0, 2*\pi] nein 0
z double Grad in Bogenmaß um den um die z-Achse rotiert werden soll: z \in [0, 2*\pi] nein 0

Rückgabewert

Typ: `tl.threed.core:Transformation`

Beispiele

Beispiel 1

rotate(Math.PI, Math.PI, Math.Pi/2);

Ausgabe: Eine Transformation die ein Objekt erst um 90° um die z-Achse dreht und dann um je 180° an der y-Achse und der x-Achse.
