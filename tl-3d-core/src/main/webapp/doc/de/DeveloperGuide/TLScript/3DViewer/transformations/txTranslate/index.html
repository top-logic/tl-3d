Syntax

translate($x, $y, $z)

Beschreibung

Erstellt eine Transformation die ein Objekt verschiebt

Parameter

Name Typ Beschreibung Pflicht Default
x double Wert um den das Objekt in x-Richtung verschoben werden soll. ja
y double Wert um den das Objekt in y-Richtung verschoben werden soll. ja
z double Wert um den das Objekt in z-Richtung verschoben werden soll. ja
